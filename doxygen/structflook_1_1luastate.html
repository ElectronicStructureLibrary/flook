Detailed Description

This type is the back-bone of the flook methodology. It contains a reference to a LUA environment which is the equivalent of a LUA shell embedded in the application. This lets you create several LUA environments simulatenously and/or open close new/old ones, at will.

It provides several class procedures which enables direct interaction with the LUA environment as well as the creation of a flook::luaTbl.

Usage

As an example we will create a new LUA instance and close it

type(luastate) :: lua
! Create a new lua environment
call lua_init(lua)
! At this point you can interact with lua through
! the other procedures in the state.
! Close lua, and create a new one (clears the lua stack
! for a fresh restart)
call lua_init(lua)
! Do some operations...
! ... and finally close it
call lua_close(lua)
